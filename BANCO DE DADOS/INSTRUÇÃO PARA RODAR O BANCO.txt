PASSO 01:

Dentro do pgAdmin:

CREATE DATABASE "Nexus_DB";


PASSO 02:

No terminal:

RESTAURANDO ESTRUTURA DO BANCO
psql -U postgres -d "Nexus_DB" -f estrutura.sql

RESTAURANDO DADOS
psql -U postgres -d "Nexus_DB" -f banco_com_dados.sql


---------------------------------------------------------------
Após finalizar modificações no banco, rodar os seguintes comandos no terminal da pasta onde se encontra o banco:

pg_dump -s -U postgres "Nexus_DB" > estrutura.sql

pg_dump -s -U postgres "Nexus_DB" > banco_com_dados.sql
