# Variáveis de ambiente para Docker

Copie este bloco para um arquivo chamado **`.env`** na **raiz do projeto** (mesma pasta do `docker-compose.yml`).

No PowerShell (Windows):
```powershell
cd "C:\caminho\para\NEXUS"
New-Item -Path .env -ItemType File -Force
# Abra .env no Notepad e cole o conteúdo abaixo
```

Ou crie manualmente o arquivo `.env` e cole:

```env
# Banco de dados (PostgreSQL no container)
POSTGRES_DB=nexus
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
DB_PORT=5432

# Portas expostas no seu computador
BACKEND_PORT=8080
FRONTEND_PORT=3000

# URL do backend usada pelo frontend no navegador
VITE_API_URL=http://localhost:8080
```

Ajuste `POSTGRES_PASSWORD` e as portas se quiser. Depois suba com:

```bash
docker compose up -d
```
